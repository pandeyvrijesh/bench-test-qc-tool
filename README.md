# Bench Test QC Automation Tool

This project is a Python-based automation tool designed to streamline the bench testing workflow in production environments.

## 🔧 Features

- Automatic screenshot detection
- Popup-based device ID tagging
- Organized local storage (device-wise)
- Google Drive integration for backup
- Used in real production workflow

## 🛠 Tech Stack

- Python
- Tkinter (GUI)
- Watchdog (file monitoring)
- File system automation

## 📂 Workflow

1. Screenshot is taken
2. Popup appears to enter Device ID
3. Screenshot is saved in device folder
4. On Finish QC → all screenshots uploaded to Google Drive

## 🚀 Real-World Usage

This tool is actively used in a production environment to:
- Track device testing
- Organize QC data
- Reduce manual work
